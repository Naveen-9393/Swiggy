#Swiggy Category-Wise Food Items, Pricing & Total Revenue Analysis – Swiggy (MySQL Project)


## key mentrics
This project focuses on analyzing food delivery data from Swiggy using MySQL. The main objective of this project is to extract meaningful business insights from raw order data and support data-driven decision-making.

In this project, I worked with datasets containing order details such as restaurant name, city, state, order date, dish name, category, price (INR), ratings, and rating count. I performed data cleaning by identifying and removing duplicate records using ROW_NUMBER() with Common Table Expressions (CTEs).

*I implemented various MySQL concepts including:

*Data Retrieval: SELECT, WHERE, ORDER BY

*Aggregation: SUM(), COUNT(), AVG(), MAX(), MIN()

*Grouping: GROUP BY, HAVING

*Joins: INNER JOIN for combining multiple tables

*Window Functions: ROW_NUMBER() for duplicate removal

*Subqueries & CTEs for advanced filtering and analysis
